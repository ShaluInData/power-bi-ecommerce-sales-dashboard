#  Ecommerce Sales Dashboard (Power BI)
Power BI dashboard analyzing ecommerce sales performance using Orders and Details datasets.

## Project Overview

This project presents an interactive Power BI dashboard analyzing ecommerce sales performance.
It helps track revenue, profit, quantity sold, and category-wise performance.

## Tools Used

* Power BI
* Excel / CSV
* Data Cleaning
* Data Visualization

## Dataset

* Orders.csv
* Details.csv

## Key Insights

* Identified top-performing product categories
* Analyzed profit and sales trends
* Monitored customer purchase behavior
